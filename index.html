My Work

Derby Days Philantrophy Platform

This is a full-stack web app for Sigma Chi’s annual fundraiser, built with Next.js, Prisma, Auth.js, Stripe, Cloudinary, and PostgreSQL. It supports secure Google sign-in, Stripe Checkout donations, and a real-time fantasy draft–style system, plus an admin dashboard with role-based access—scheduled for deployment in Spring 2026 to track donations and report results live.

NFL Database

A fully normalized relational database (3NF) built from multiple CSV and JSON datasets covering NFL players, teams, games, and stats. All data was cleaned, transformed, and loaded into a PostgreSQL database hosted on Neon, where it can be queried directly. The schema was designed to eliminate redundancy while preserving complex relationships across the league's historical data.

Study Buddy

A college-focused social media platform built as a group project, where I served as project manager and lead engineer. Students join their school's community and get matched with classmates based on shared courses and interests. Features include a real-time live chat similar to iMessage, a professor rating system similar to Rate My Professor, AI-powered summaries, and full profile creation and editing, giving students one place to connect, review, and collaborate.

Fingerprint Authentication System

This is a Python-based biometric security project designed to verify users through fingerprint analysis. Built with OpenCV and NumPy, it processes fingerprint images through normalization, binarization, and skeletonization before extracting unique minutiae points using Crossing Number and gradient-based methods. The system matches fingerprints with rotation and translation alignment, achieving over 99% accuracy on large-scale datasets while minimizing false acceptance and rejection rates.

Blackjack Monte Carlo Simulation

A Python simulation that runs thousands of blackjack hands to compare the performance of different strategies, including basic strategy, random play, and custom approaches. Results are visualized through matplotlib graphs showing win rates over time. Also includes a fully playable command-line version of the game.

Cellular Automata Simulation

A Python simulation of two cellular automata: Conway's Game of Life and the Star Wars ruleset, both rendered and animated with matplotlib. Each simulation evolves on a grid where cells live or die based on their neighbors, producing complex emergent patterns from simple rules.
