It supports secure Google sign-in, Stripe Checkout donations, and a real-time fantasy draft–style system, plus an admin dashboard with role-based access—scheduled for deployment in Spring 2026 to track donations and report results live.

Fingerprint Authentication System

This is a Python-based biometric security project designed to verify users through fingerprint analysis. Built with OpenCV and NumPy, it processes fingerprint images through normalization, binarization, and skeletonization before extracting unique minutiae points using Crossing Number and gradient-based methods. The system matches fingerprints with rotation and translation alignment, achieving over 99% accuracy on large-scale datasets while minimizing false acceptance and rejection rates.
